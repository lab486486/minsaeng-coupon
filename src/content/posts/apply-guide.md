## 신청 전 체크리스트

- [ ] 주민등록상 거주지가 신청 지자체와 같은지
- [ ] 본인 명의 휴대폰·공동인증/간편인증 준비
- [ ] 받고 싶은 지급 수단(지역화폐 vs 카드 포인트) 결정
- [ ] 사용 기한·사용처 제한(대형마트·백화점 제외 여부 등) 확인

## 일반적인 신청 흐름

1. **공고 확인** — 시·군·구 홈페이지, 정부24, 지역 신문·문자
2. **온라인 신청** — 정부24 / 지자체 전용 페이지 / 지역화폐 앱
3. **지급** — 신청 후 수일~수주 내 충전 또는 포인트 반영
4. **사용** — 가맹점에서 기한 내 사용

## 자주 헷갈리는 점

- **전국민 자동지급이 아닐 수 있음** → 신청해야 받는 방식이 흔함  
- **타 지역 사용 제한** → 지역화폐형은 관내 사용이 원칙인 경우가 많음  
- **중복 수급** → 중앙·지자체 사업이 겹치면 하나만 가능하거나 차감될 수 있음
